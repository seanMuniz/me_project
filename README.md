# me_project
This project showcases my experiences in the Canadian Armed Forces. This project also showxases my design approach to making
a layout for showcasing images and text. This project also demonstrated use of flex properties to layout elements. Lastly,
This project also demonstrates responsiveness, as the layout changes in accordance to the size of the screen.
